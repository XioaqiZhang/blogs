为什么需要预冲电路

如图所示，动力电池所带的电机控制器负载，在前端都有一个大电容C，在冷态启动时，C上没有电压或只有很低的残留电压，当没有预充电时，主接触器K+、K-直接与C接通，此时电池电压VB有高压，而C上的电压接近0，相当于瞬间短路，此时的负载仅仅是导线和继电器触点的电阻，一般远小于20mΩ。根据欧姆定律，回路电阻按20mΩ计算，VB和VC之间的压差按300V计算，则瞬间电流I = 300 / 0.02 = 15000A，大电流必然导致继电器损坏。

预冲的解决方案

加入预充电过程，在上电过程中，先不闭合K+，而是先将预冲继电器Kp闭合，预冲电阻一般选用100欧姆到200欧姆，VC与VB之间的压差仍然按300V计算，在接通瞬间的电流I= 300/200 = 1.5A，所以预冲回路是安全的。
预冲电路工作后，大电容C上的电压越来越大，VB与VC之间的压差越来越小，当VC足够接近VB时，切断预冲继电器Kp，接通主接触器K+，此时就不存在大电流冲击，因为I= （VB - VC）/R。
车辆上的接触器开关状态由VCU控制，一般上电过程预冲的处理有两种方案：

1.接通K-和预冲继电器Kp，检测电机控制器母线端电压（VC）与动力电池电压（VB）的压差，当这个差值足够小，切断Kp，接通主接触器K+。
2.接通K-和预冲继电器Kp，延时一段时间t，然后切断Kp，接通主接触器K+。
那么时间t应该如何计算呢？
假设：
V0为电容上初始电压值
V1为电容最终可充/可放到的电压值
Vt则为t时刻电容上的电压值
那么符合公式：
Vt = V0 + （V1 - V0）[1-exp(-t/RC)]
或
t = RCLn[(V1-V0)/(V1-Vt)]

根据以上公式，假设我们要在VB为VC的90%（V0 = 0, V1= VC, Vt = 0.9VC）时结束预冲，闭合主接，那么预冲持续的时间t = RC * Ln(10)=2.3RC，
常用的R=200Ω，C=500μF，带入t= 230ms

或者有一种更简单的计算办法，RC回路的时间常数τ = R C，在充电时，每经过一个τ时间，电容器上的电压就上升0.632倍的(VB - VC)，那么粗略计算大约2τ~3τ左右的时间就可以让VB接近VC的90%了，也就是安全状态了，同样带入上面的数据：t=2.5τ = 2.5R*C=250ms
